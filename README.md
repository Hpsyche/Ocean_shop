## 优点与不足
### 优点
* 市场潜力是巨大的
* 劳动力成本优势是显而易见的
* 方便和可靠供应
* 强大的宣传工具

## 不足
* 由于资源有效，部分背景图较为模糊。
* 缺少意见反馈模块。

## 怎么改进
* 加大检索力度，多渠道找到高清的海大美图。
* 增加此模块，让访客能够对此网站提出宝贵意见，也有利于网站的进一步改进。
